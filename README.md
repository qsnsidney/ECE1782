# Tiny Universe Simulator (TUS)

## Executables

### tus (main project)
- The GPU simulator written in CUDA
- Located in ./src/tus/

### cpu_sim
- The CPU simulator
- Located in ./src/cpu_sim


## Makefile

### tus
```
# Compile only
make tus
# Compile and run
make run_tus
```

### cpu_sim
```
# Compile only
make cpu_sim
# Compile and run
make run_cpu_sim
```
